# shopbgremover

AI product background remover for Shopify sellers.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to start.

## Tech Stack

- Next.js 14
- Cloudflare Pages
- Remove.bg API
- Cloudflare D1 (Database)
- Google OAuth
- PayPal
